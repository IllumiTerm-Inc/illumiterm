 <h2 align="center">IllumiTerm</h2>

<p align="center">
  <img src="https://github.com/IllumiTerm-Inc/illumiterm/blob/main/data/icons/about.png" alt="Screenshot">
</p>

Built with GTK3 and VTE 2.91 is a sleek and feature-packed terminal emulator designed to enhance productivity and streamline your workflow. With a minimalist yet powerful interface, it integrates all the essential terminal features alongside advanced functionalities like multiple tabs, customizable keybindings, and smooth navigation.

The terminal window allows users to easily manage their workspace with intuitive tab handling and fast switching between sessions. Designed with efficiency in mind, you can create new windows, switch between tabs, zoom in or out, and even enter fullscreen mode—all with quick keyboard shortcuts.

Beyond just a standard terminal, the app incorporates user-friendly menu items like "Preferences", "Help", and context-aware menus for easy access to commonly used actions such as copy, paste, or clearing the scrollback buffer. Whether you are a developer, system admin, or a power user, this terminal provides a responsive, polished environment to run your commands, with the added bonus of full customization through Python.

It’s a powerful yet elegant solution, built to handle complex workflows, all while giving you the flexibility and control of a custom terminal experience!

## On Debian, Ubuntu or their derivatives

```
$ sudo make install
```
